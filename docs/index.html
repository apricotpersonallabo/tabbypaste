About Tabby Paste

Tabby Paste is a lightweight Chrome extension that helps you copy and paste data efficiently between web pages and applications. Whether you're working with tables, forms, or structured data, Tabby Paste makes the process seamless and intuitive.

📋

Easy Data Copy

Click any row in a table to copy data in TSV format, ready to paste into spreadsheets and forms.

⚡

Lightweight & Fast

No external dependencies or heavy libraries. Pure JavaScript for lightning-fast performance.

🎨

Simple Interface

Intuitive design that works with any website. Just click on the data you want to copy.

How It Works

1

Install

Add Tabby Paste to your Chrome browser from the Chrome Web Store.

2

Use

Click on any table row or data element to automatically copy it in TSV format.

3

Paste

Paste the copied data into your spreadsheet, form, or any other application.

Install

Install Tabby Paste from your browser's extension store:

  • Chrome Web Store
  • Microsoft Edge Add-ons
